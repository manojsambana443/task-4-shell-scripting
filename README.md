# Task 4: Shell Scripting Basics

## Objective
The goal of this task is to understand shell scripting fundamentals and automate repetitive DevOps-related tasks using Bash.

## Tools Used
- Bash Shell
- Linux Terminal

## Scripts Included

### 1. system_info.sh
Displays system details like hostname, OS information, uptime, current user, and disk usage.

### 2. loop_demo.sh
Demonstrates the use of loops and conditionals in Bash.

### 3. log_cleanup.sh
Automatically deletes log files older than 7 days from a test directory.

### 4. backup.sh
Creates a compressed backup of the Documents folder.

Make scripts executable:
